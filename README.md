# PRODIGY_DS_01

## Task 01 - Data Visualization

### Objective

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Dataset

World Population Dataset (World Bank)

### Tools Used

* Python
* Pandas
* Matplotlib

### Steps Performed

1. Loaded and cleaned the dataset.
2. Extracted population data for the latest available year.
3. Selected the top 10 countries by population.
4. Created a bar chart to visualize population distribution.
5. Saved the visualization as an image.

### Visualization

* Top 10 Countries by Population (Bar Chart)

### Outcome

The visualization highlights the countries with the highest population and demonstrates how data visualization can be used to understand population distribution patterns.
